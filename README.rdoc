== README

Good tracker gives you a picture of where your time is spent at weekly / monthly basis and
extrapolates the info to say where you ll be in a year.

Tech stack :

1. Ruby on Rails
2. Chartkick
3. Postgres

Project is in active development and you can easily clone the project and run the application using ./run script. Use `./run help` for more info.

<tt>rake doc:app</tt>.
